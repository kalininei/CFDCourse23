Метод конечных разностей
1. Методы решения уравнения диф. уравнения первого порядка du/dx = f(x)
   Метод Эйлера, Лакса-Вендорффа, Кранка-Николлсона

2. Аппроксимация уравнения Пуассона -d^2(y)/dx^2 = f(x).
Учёт граничных условий: первого, второго, третьего рода. Периодические граничные условия.

Метод конечных объёмов
1. Общие понятие метода конечных объёмов. Конечнообъёмная сетка и способы её хранения.
   Разряженная система линейных уравнений и CSR-формат её хранения.

2. Аппроксимация уравнения Пуассона -d^2(y)/dx^2 = f(x) в 1/2/3D областях

3. Постановка граничных условий для уравнения Пуассона.

4. Учёт внутренних источников и стоков при написании конечнообъёмной схемы уравнения Пуассона.
   Аппроксимаци уравнения вида -d^2(y)/dx^2 + e(x)*y(x) = f(x).

Метод конечных элементов
1. Общие принципы построения конечноэлементных схем: метод взвешенных невязок, метод Галёркина, ослабление постановки.
2. Принцип построения базисных функций. Параметрическое и физическое пространства. Матрица Якоби.
   Формулы интегрирования и дифференцирования.
3. Принцип сборки глобальных матриц черех набор локальных матриц для каждого элемента.
   Аналитическое и численное вычисление интегралов.
4. Узловые линейные базисные функции в симплексных элементах в 1/2/3 D
5. Билинейный четырёхуглольный элемент в 2D. Принцип построения узловых базисных функций высокого порядка.
6. Постановка граничных условий 1, 2 и 3-его рода в конечноэлементной расчётной схеме при решении уравнения Пуассона.
